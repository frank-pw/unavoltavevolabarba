# Portfolio design

Creare una pagina statica `portfolio.html` nello stile nero e minimale del sito.

- Navigazione: Home, Blog, Portfolio, Contatti.
- Contenuto: titolo “Portfolio” e due grandi schede per `deep-cat-blue` e `cordyceps`.
- Ogni scheda usa una copertina astratta CSS e collega al repository GitHub in una nuova scheda.
- Layout: due colonne su desktop, una colonna su mobile.
- Nessuna API GitHub, dipendenza o JavaScript specifico della pagina.
- Verifica: presenza della pagina, ordine della navigazione e URL esatti dei repository.
